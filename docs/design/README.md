# Design mockups

`dashboard.html` — high-fidelity, self-contained interactive mockup of the
FlashSentiment Trader dashboard (single screen: animated news→order pipeline +
cockpit as home, plus a Sources catalog and volatility-ranked Calendar whose
armed events fire through the same pipeline). Clean light SaaS; theme-aware.

It is a **design reference** (client-side simulated data), not wired to the
backend. Open it directly in a browser. The spec it implements is in
[`../design-brief.md`](../design-brief.md).
